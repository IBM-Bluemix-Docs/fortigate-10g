# 取消防火牆

FortiGate Security Appliance 10Gbps 可以立即或在週年日期取消。在週年日期取消的防火牆仍會保持作用中，直到下一個收帳期間開始為止。請採取下列步驟來取消防火牆：

1. 選取**防火牆名稱**直欄之下的防火牆名稱以導覽至「裝置概觀」頁面。
2. 按一下溢位功能表，然後選取**取消裝置**。
3. 選擇立即取消或者在週年日期（下一個計費週期）取消。
4. 確認您選取的項目。

**附註：**不支援從高可用性降級至獨立式伺服器。
