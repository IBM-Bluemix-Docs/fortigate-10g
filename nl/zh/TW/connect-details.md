# 可用的資料中心
{: #available-data-centers}

IBM© 提供許多全球資料中心，以便連接您的 Fortigate Security Appliance (FSA) 的 IBM Cloud 部署。我們提供下列速度的連線：

* 50 mbps
* 100 mbps
* 200 mbps
* 1 Gbps
* 2 Gbps
* 5 Gbps

每個 IBM Cloud 資料中心都連接至 IBM 全球專用網路 — 讓各地的資料傳輸更快速且更有效率。
